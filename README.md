Dengan adanya web keuangan yang saya buat dengan bantuan chat AI dari Chat GPT dan juga dari Gemini.
Harapan saya dari repositori yang saya buat ini ada yang bisa memberikan saran dan masukan agar bisa lebih menarik kembali, jika kamu mau coba, 
pastikan sudah mempunyai akun supabase dan sudah membuat table seperti transaksi dan users. 
Untuk kolom table *transaksi* yaitu *id* datatype int8, *tanggal* datatype date, *jenis* datatype text, *kategori* datatype text, *jumlah* datatype numeric, *keterangan* datatype text.
Untuk kolom table *users* yaitu *id* datatype int8, *username* datatype text, *email* datatype text, *phone* datatype text, *password* datatype text.
Pastikan untuk Row Level Security (RLS) disable agar bisa diakses oleh HTML yang saya buat.
Lalu sesuaikan url dan API key supabase kita buat dilampirkan ke script.js.
